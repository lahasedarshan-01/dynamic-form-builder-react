# Dynamic Form Builder (React.js)

A Dynamic Form Builder built using React.js that allows users to create customizable forms without writing code.

## Features

- Dynamic Form Creation
- Add/Delete Fields
- Edit Labels & Placeholders
- Dynamic Dropdown, Radio & Checkbox Options
- Form Validation
- Live Preview
- Dark Mode
- Save Template (Local Storage)
- Export JSON

## Tech Stack

- React.js
- Bootstrap 5
- JavaScript (ES6)
- HTML5
- CSS3

## Installation

npm install

npm start
